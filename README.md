## 🎯 Overview

**VietHay** is an intelligent AI-powered video generation platform designed specifically for Vietnamese e-commerce sellers on Shopee, TikTok Shop, and Lazada.

It combines the power of **TRAE SOLO** (advanced AI coding agent) and **PixVerse** to help sellers create high-quality, culturally relevant product videos at lightning speed.

### The Problem
Vietnamese sellers spend hours (or pay high costs) to create engaging product videos. This slows them down and makes it hard to keep up with fast-moving trends.

### The Solution
Just describe your product → **VietHay** automatically generates a compelling script, storyboard, and a **30+ second cinematic video** optimized for Vietnamese audiences.

---

## ✨ Key Features

- **Text-to-Video Generation**: Input product name + description → Full video in ~60 seconds
- **Vietnamese Cultural Intelligence**: Localized storytelling, humor, FOMO, trendy Vietnamese vibes
- **Smart Script & Storyboard Engine**: Powered by TRAE SOLO
- **Built-in Video Editor**: Quick subtitle editing + mock dubbing
- **A/B Testing & Mock Analytics**: Compare versions and see simulated performance
- **Video History Gallery**: Save and manage all generated videos
- **Fully Responsive**: Excellent experience on both desktop and mobile

## 🚀 How It Works

1. Enter product details (name, description, style)
2. TRAE SOLO generates optimized script + PixVerse-ready prompts
3. PixVerse generates high-quality video
4. Edit subtitles, download, or iterate instantly
