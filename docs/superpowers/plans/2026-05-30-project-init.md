# VietHay Project Initialization Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `executing-plans` to run these steps sequentially (this session is doing inline execution).

**Goal:** Khởi tạo codebase VietHay bằng Next.js 15 + TypeScript + Tailwind + shadcn/ui, sẵn sàng build UI theo phong cách TRAE.

**Architecture:** Monorepo đơn giản: tài liệu ở root, app Next.js nằm trong `web/`. Các phần còn lại sẽ được triển khai theo plan tổng.

**Tech Stack:** Next.js 15 (App Router), React, TypeScript, Tailwind CSS, shadcn/ui.

---

## Task 1: Scaffold Next.js app

**Files/Dirs:**
- Create: `web/` (generated by create-next-app)

- [ ] **Step 1: Tạo Next.js app**
  - Run (tại thư mục root dự án):
    - `npx create-next-app@latest web --ts --tailwind --eslint --app --src-dir --import-alias "@/*" --use-npm`
  - Expected: tạo thư mục `web/` với Next.js App Router + Tailwind.

- [ ] **Step 2: Chạy dev server để smoke test**
  - Run:
    - `npm run dev --prefix web`
  - Expected: app chạy ở `http://localhost:3000`

---

## Task 2: Initialize shadcn/ui

**Files/Dirs:**
- Modify/Create (trong `web/`): `components/ui/*`, `tailwind.config.*`, `app/globals.css`, `components.json`

- [ ] **Step 1: Xem options của shadcn CLI (tránh interactive)**
  - Run:
    - `npx shadcn@latest init --help`

- [ ] **Step 2: Init shadcn/ui (defaults)**
  - Run (trong `web/`):
    - `npx shadcn@latest init --yes`
  - Expected: tạo `components.json`, cấu hình Tailwind, globals.

- [ ] **Step 3: Add base components cho layout**
  - Run:
    - `npx shadcn@latest add button card input textarea tabs dialog dropdown-menu badge`
